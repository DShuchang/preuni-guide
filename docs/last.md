# 后记

我很庆幸自己能从广西的一个小县城来到上海求学。来到这里之后，我看到了更广阔的世界，也感受到了许多来自外界的善意与帮助。

让我印象很深的是，在我准备到上海之前，有很多来自恭中长三角校友会的学长学姐私下联系我，帮我接机、安排酒店和临时休息的地方。这让我非常感动，也让我真切感受到：从同一个小地方走出来的人，其实也可以彼此支持、彼此照亮。

所以我想写这样一份指南，送给我的学弟学妹们。希望它能在你们高考结束后、填报志愿前，以及正式进入大学之前，少一些迷茫，多一些确定感。哪怕这篇文章只能帮到你们一点点，我也会很开心。因为这也算是把我从恭城中学得到的温暖，继续传递给后来的人，同时，我也希望你们以后在力所能及的时候，也能去帮助自己的学弟学妹们。

我很喜欢博主 Tairan 的频道 [WhynotTV](https://space.bilibili.com/14145636)，也很喜欢他与翁家翌学长在 [Podcast](https://www.bilibili.com/video/BV1darmBcE4A/) 中关于开源精神的分享。那期节目让我印象很深，也让我更加认同开源的意义：把自己整理过的东西分享出来，让后来的人少走一些弯路。

同时，也感谢殷旻学长的 [CSDIY Wiki](https://csdiy.wiki/)。它让我了解到海内外许多优秀的计算机学习资源，也启发我思考：我是不是也可以整理一份面向准大学生的指南。

很感谢大家能够读到这里。这是我第一次自己搭建一个网站，也算是一个小小的项目。希望我的学弟学妹们在走向大学校园时，能够少一些迷茫，多一些勇气，也能够更加积极、坚定地迎接属于自己的大学生活。
