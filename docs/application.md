# 志愿填报总览

高考结束后，大家很容易被各种志愿填报信息淹没：多少分能报什么学校，哪些专业热门，哪些专业不能选，志愿应该怎样安排“冲、稳、保”……这些信息有一定参考价值，但观点往往相互矛盾，最终还是要根据自己的情况作出判断。

填报志愿时，首先需要明确个人的基本条件和限制，包括成绩位次、选科要求、体检条件、家庭经济情况，以及自己能够接受的城市、学费和学习方式。在这些条件之内，再比较专业、学校和城市。

兴趣是选择专业时的重要因素，但不能只根据专业名称或模糊印象判断。真正需要了解的是：这个专业本科阶段学习哪些课程，需要怎样的学科基础，日常学习更偏向实验、编程、背诵还是实践，以及毕业后通常有哪些发展方向。


对专业有一定兴趣，通常更容易形成学习动力，但兴趣并不意味着学习过程始终轻松。即使暂时没有非常感兴趣的方向，也不要简单选择所谓的“热门专业”。行业热度会变化，当前热门不代表几年后仍然热门，受到政策支持的行业也不代表其中所有岗位都适合自己。相比追逐热度，更重要的是确认自己能否接受专业的学习内容，并了解它是否具有多种发展路径。


学校层次、专业匹配度和所在城市之间没有适用于所有人的固定排序。有人更重视学校平台和升学机会，有人更看重专业实力，也有人希望尽早接触相关行业。填报前可以先问自己：在学校、专业和城市之间，自己最不能接受的结果是什么，又最愿意为哪一项作出妥协？

一些学校的部分专业录取位次可能明显低于学校整体水平，但不要只把这种情况理解为“低分冲名校”的机会。天下没有免费的午餐，分数较低带来的可能是较为冷门小众的专业，在想要低分冲名校前一定要判断自己是否都能接受就读这些冷门的专业。
尤其是在以专业组为单位填报的地区，不要只关注专业组里自己最喜欢的专业。需要认真查看组内的全部专业，并按照最不理想的录取结果进行判断。如果自己无法接受其中某个专业，就要谨慎考虑是否填报这个专业组。不要把“入学后再转专业”当作必然的退路，因为转专业通常会受到成绩、名额和考核条件的限制。同时，以前几年广西的报考情况来看请大家务必要勾选服从调剂这个选项，如果不勾选的话，假如你的分数线高于该校的最高投档线但是你的分数线低于你在这个学校填报的所有专业的录取分数线且没有服从调剂的话，你会被该大学退档，滑档到下一个档次（比如本科批就会滑到专科批），所以切记一定要勾选上服从调剂。
大学所在城市也值得考虑。大城市通常拥有更多企业、实习和行业活动，但生活成本和竞争压力也更高；一些非一线城市则可能在特定产业上具有明显优势。与其简单比较城市等级，不如了解当地是否有与专业相关的产业、学校是否具备校企合作资源，以及自己是否愿意长期在当地学习和生活。
志愿填报涉及录取规则的内容，应以本省教育考试机构、高校招生官网和招生章程为准。重点查看招生计划、往年录取位次、专业选科要求、体检限制、单科成绩要求、收费标准和调剂规则。尽量不要使用传统的冲稳保策略，而是在客观理性分析后判断自己的分数能录哪些大学哪些专业，在此基础上面再来填志愿，这样会尽可能被自己能够接受的志愿录取。


最后，我想分享一下自己当时填报志愿的过程。这并不一定适用于所有人，但其中的一些思路可以作为参考。
高考结束后，我先花了大约十天时间了解不同专业的培养内容和发展方向，逐渐将自己的意向范围缩小到航空航天、自动化和计算机三个方向。在正式出分前，我根据平时模拟考试的成绩和市排名，粗略估计了自己高考成绩可能对应的全省位次范围。我当时使用的方法，是结合本市考生规模，将模拟考试的市排名乘以大约4至5.5，得到一个比较宽的全省位次区间。需要说明的是，这只是我根据当时所在地区的情况采用的个人估算方法，不是适用于所有地区的通用公式。不同城市的考生规模、模拟考范围和成绩分布差别很大。出分前进行估算的目的，也不是准确预测自己的排名，而是提前建立一个大致的院校范围。确定预估位次区间后，我把这个范围内可能报考的学校尽量列出来，再根据意向专业、所在城市、学校的优势学科和培养特点等因素进行第一轮筛选。对于剩下的学校，我进一步整理了2022年至2024年相关专业或专业组的录取位次、招生计划及其波动情况，并制作表格进行比较。根据这些资料，我会估计2025年可能出现的录取位次区间，而不是只预测某一个具体名次。实际录取位次会受到招生计划、专业组调整、报考热度和所谓“大小年”等因素影响，因此往年数据只能用于判断风险，不能当作精确答案。出分后，我再根据自己的实际全省位次删去明显不合适的学校，并对剩余院校进行排序。排序时，我不仅考虑进入学校的可能性，也会考虑自己能否接受专业组内的其他专业、学校所在城市、专业实力以及被调剂后的最差结果。经过这些步骤，最终形成自己的志愿方案。此外，我非常建议大家关注当地在出分前后举办的高校招生咨询会。桂林往年会举办类似活动，去年是在甲天下广场，参加咨询会的高校范围通常比较广，从普通高校、部分“211”高校到顶尖985都可能设置咨询展位。与招生老师面对面交流，可以获得许多仅靠网上搜索不容易判断的信息。例如，当年的招生计划是否变化，专业组怎样设置，某个位次报考学校或专业的风险大致如何，以及不同专业实际学习哪些课程。招生老师往往比考生更熟悉本校历年的生源位次和当年的咨询情况，因此他们的判断很有参考价值。不过，这种判断也不是录取承诺。每年的报考人数、志愿分布和招生政策都可能发生变化，招生老师同样无法提前准确预测最终结果。我当时在线下咨询时，曾分别与复旦和中科大的招生老师交流。他们了解我的位次和意向后，直接建议我去咨询上海交大而不是报复旦或者中科大。正是因为这个建议，我进一步了解了交大的培养方案和专业选择。后来，在与上海交通大学招生组的交流中，我逐渐明确了自己对计算机方向的兴趣，并最终作出了相应选择。对我而言，这些交流不仅提供了录取位次方面的参考，也让我对专业学习内容和学校培养方式有了更具体的认识，而这些信息仅靠网上浏览很难完全获得。

没有条件参加线下咨询会的同学，也可以主动联系目标高校的招生老师。学校招生官网、官方公众号或学院通知中，通常会公布各地区招生咨询联系方式。
打电话前，建议先整理好自己的省份、选科、全省位次和意向专业，并提出具体问题。例如：
我的位次报考贵校大致处在什么风险区间？
这个判断是针对学校投档，还是某个具体专业？
今年的招生计划和专业组设置是否发生变化？
专业组内是否存在我需要特别注意的专业？
是否建议服从调剂，不服从调剂可能有哪些风险？
相关专业主要学习什么，入学后能否转专业？

招生老师的意见不能替代招生章程和官方录取数据，但它可以帮助我们更准确地理解学校当年的招生情况，也能发现自己原本没有注意到的信息。志愿填报时，最好把历年数据、官方政策、招生老师的意见和自己的实际需求结合起来，而不是只依赖其中某一种信息。
志愿填报很难找到一个各方面都完美的选项。更现实的目标，是在充分了解信息的基础上，选择一个自己愿意接受、也能够承担其风险的方案。
