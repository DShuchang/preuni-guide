## <a id="toc-1"></a>一、三个专业一句话区分

**人居设计**：主要研究人类居住、生活、工作、交往、出行、休闲等综合环境如何被系统设计，核心关键词是**未来人居、空间设计、城乡更新、环境体验、生活方式、生态低碳、智慧空间、设计治理**。

**工业设计**：主要研究产品从功能、结构、外观、人机交互、材料工艺到用户体验的整体设计，核心关键词是**产品设计、用户体验、结构工艺、人机工程、智能硬件、交通工具、医疗产品、服务设计**。

**视觉传达设计**：主要研究如何通过文字、图形、色彩、版式、影像、品牌和数字界面进行信息表达，核心关键词是**品牌设计、平面设计、海报、字体、包装、UI、信息设计、广告视觉、新媒体视觉**。

简单说：

- **人居设计**更像是“设计人与环境、空间、城市、社区、生活方式之间的关系”
- **工业设计**更像是“设计产品如何好用、好看、好制造、好体验”
- **视觉传达设计**更像是“设计信息如何被看见、理解、记住和传播”

也可以这样理解：

> 人居设计：偏空间、人居环境、未来生活方式和系统设计  
> 工业设计：偏产品、硬件、用户体验和工程制造  
> 视觉传达设计：偏视觉表达、品牌传播、平面与数字界面

## <a id="toc-2"></a>二、专业代码与学科定位

| 专业 | 本科专业代码 | 所属门类 | 所属专业类 | 常见授予学位 | 简要定位 |
|---|---|---|---|---|---|
| 人居设计 | 130514T | 艺术学 | 设计学类 | 艺术学学士 | 新兴设计专业，偏人居环境、空间系统、生活方式和设计治理 |
| 工业设计 | 080205 | 工学 | 机械类 | 工学学士 | 工程与设计交叉，偏产品、制造、人机、体验和工程转化 |
| 视觉传达设计 | 130502 | 艺术学 | 设计学类 | 艺术学学士 | 经典设计专业，偏图形、品牌、版式、包装、界面和视觉传播 |

说明：

- **人居设计**是 2025 年列入本科专业目录的新专业，专业代码中的 `T` 表示特设专业。
- **工业设计**在本科目录中属于工学门类下的机械类，和机械、材料、制造、产品工程、人机工程关系密切。
- **视觉传达设计**属于艺术学门类下的设计学类，是传统设计类中就业面较广的专业之一。
- 很多学校还开设与这三个专业相近的方向，例如环境设计、产品设计、数字媒体艺术、艺术与科技、建筑学、城乡规划、交互设计、服务设计等。报考时一定要看具体学校的培养方案和课程设置。

## <a id="toc-3"></a>三、三个专业核心区别总览

| 维度 | 人居设计 | 工业设计 | 视觉传达设计 |
|---|---|---|---|
| 学科属性 | 艺术学、设计学、新兴交叉 | 工学、机械类、设计工程交叉 | 艺术学、设计学、视觉传播 |
| 核心对象 | 人居环境、空间系统、生活方式 | 产品、硬件、工具、设备、服务系统 | 图形、文字、品牌、版式、界面、信息 |
| 核心问题 | 人如何更好地生活在空间与环境中 | 产品如何更好用、更好看、更好制造 | 信息如何更有效地被看见和理解 |
| 主要成果 | 空间方案、场景方案、城市更新方案、体验系统 | 产品模型、产品方案、结构草图、交互原型 | 品牌视觉、海报、包装、字体、UI、视觉系统 |
| 更偏空间还是产品 | 空间和系统 | 产品和体验 | 信息和视觉 |
| 工程要求 | 中等 | 高 | 中等偏低 |
| 美术基础要求 | 高 | 中高 | 很高 |
| 软件建模要求 | 高 | 高 | 中高 |
| 用户研究要求 | 高 | 高 | 中高 |
| 商业转化要求 | 中高 | 高 | 高 |
| 本科就业直接性 | 中等，新专业需要靠作品和方向 | 中高，产品与制造岗位较清晰 | 中高，视觉设计岗位较多 |
| 读研价值 | 高 | 中高 | 中高 |
| 典型就业 | 空间设计、文旅更新、城市更新、展陈、生活方式设计 | 产品设计、结构协同、智能硬件、交通工具、UX、CMF | 平面设计、品牌设计、包装设计、UI、新媒体视觉、广告 |
| 最大优势 | 新兴交叉，视野宽，适合未来人居和系统设计 | 工程落地强，产品设计岗位相对明确 | 应用广，作品转化快，商业需求稳定 |
| 最大挑战 | 新专业认知度仍在形成，需要主动建立方向标签 | 需要同时懂设计、工程和制造 | 竞争激烈，容易被工具化，需要形成审美和策略能力 |

## <a id="toc-4"></a>四、三个专业的共同特点

### <a id="toc-4-1"></a>1. 都属于设计相关专业，但设计对象不同

三个专业都属于广义设计领域，但设计对象不同：

| 专业 | 设计对象 |
|---|---|
| 人居设计 | 人的生活环境、空间系统、社区场景、未来生活方式 |
| 工业设计 | 产品、设备、工具、智能硬件、交通工具、服务系统 |
| 视觉传达设计 | 图形、文字、品牌、广告、包装、界面、视觉信息 |

设计不是单纯“画得好看”。

真正的设计要考虑：

- 用户是谁
- 使用场景是什么
- 问题在哪里
- 需求如何被发现
- 方案如何被表达
- 作品如何落地
- 成本是否可控
- 材料和技术是否可行
- 体验是否合理
- 传播是否有效
- 是否符合社会、文化和商业环境

---

### <a id="toc-4-2"></a>2. 都需要作品集意识

这三个专业就业和深造都非常看重作品集。

作品集不是简单把作业拼在一起，而是要体现：

- 你的问题意识
- 你的调研过程
- 你的设计逻辑
- 你的方案推导
- 你的审美能力
- 你的表达能力
- 你的软件能力
- 你的落地意识
- 你的项目完整度

不同专业作品集重点不同：

| 专业 | 作品集重点 |
|---|---|
| 人居设计 | 空间分析、场景研究、调研地图、概念生成、空间方案、系统设计 |
| 工业设计 | 产品草图、用户研究、结构分析、CMF、三维建模、渲染、模型制作 |
| 视觉传达设计 | 品牌系统、海报、字体、版式、包装、UI、信息可视化、视觉叙事 |

---

### <a id="toc-4-3"></a>3. 都受到 AI 和数字工具影响

AI 生成图像、AIGC 设计工具、三维生成、智能排版、自动建模和低代码工具正在改变设计行业。

可能被冲击的能力包括：

- 简单海报排版
- 基础图标制作
- 普通效果图生成
- 常规产品外观草案
- 初级视觉素材制作
- 简单包装 mockup
- 基础空间氛围图

但真正有价值的设计能力仍然重要：

- 发现问题
- 定义问题
- 用户研究
- 设计策略
- 审美判断
- 文化理解
- 系统思维
- 商业判断
- 材料工艺理解
- 空间体验设计
- 产品落地能力
- 品牌一致性控制
- 复杂项目沟通

未来设计师的核心竞争力不是“不用 AI”，而是：

> 会用 AI 提高效率，同时能提出真正有价值的设计判断和解决方案。

## <a id="toc-5"></a>五、人居设计专业

### <a id="toc-5-1"></a>1. 专业特点

人居设计是面向未来生活环境的新兴设计专业。

它关注的不只是室内空间，也不只是建筑外观，而是人和生活环境之间的整体关系。

它研究的问题包括：

- 人未来如何生活？
- 社区空间如何更宜居？
- 城市更新如何更有人情味？
- 老旧空间如何重新焕发生命力？
- 居住空间如何兼顾舒适、健康和可持续？
- 文旅空间如何形成文化体验？
- 公共空间如何促进交往？
- 智慧技术如何进入人居环境？
- 老龄化社会需要怎样的居住环境？
- 低碳生活方式如何通过设计实现？
- 虚拟空间和现实空间如何融合？

人居设计的核心是：

> 以人为中心，综合设计空间、环境、生活方式、文化体验和社会关系。

---

### <a id="toc-5-2"></a>2. 专业定位

人居设计处在多个专业交叉区域：

- 环境设计
- 建筑设计
- 城乡规划
- 景观设计
- 室内设计
- 公共艺术
- 服务设计
- 交互设计
- 社区治理
- 文旅设计
- 生态设计
- 智慧城市
- 生活方式研究

它不像传统环境设计那样只关注室内外空间效果，也不像建筑学那样以建筑单体设计为主。

它更强调：

- 多尺度设计
- 多场景设计
- 多主体协同
- 生活方式研究
- 未来人居趋势
- 空间与社会关系
- 生态与低碳设计
- 智能技术与空间融合
- 文化传承与空间更新

---

### <a id="toc-5-3"></a>3. 适合人群

人居设计适合：

- 喜欢空间、建筑、城市、社区和生活方式
- 对人的居住体验和公共空间感兴趣
- 有较强审美能力
- 愿意做调研和方案推导
- 对文旅、城市更新、展陈、空间体验感兴趣
- 关注老龄化、低碳、可持续、智慧城市等议题
- 喜欢综合性设计而不是单一视觉设计
- 愿意学习建筑、环境、社会、技术和文化知识

不太适合：

- 只想做简单室内效果图
- 不愿意做调研
- 不喜欢空间尺度和场地分析
- 不愿意画图、建模和做方案表达
- 只追求视觉好看，不关注人的真实需求
- 不愿意接触跨学科内容

## <a id="toc-6"></a>六、工业设计专业

### <a id="toc-6-1"></a>1. 专业特点

工业设计是工学与设计交叉的专业，主要研究工业产品如何被设计、制造、使用和体验。

它关注的问题包括：

- 产品如何更好用？
- 产品外观如何更符合用户审美？
- 产品结构是否合理？
- 产品能不能被制造出来？
- 材料和工艺是否适合？
- 人手如何握持更舒适？
- 交互流程是否顺畅？
- 产品如何降低成本？
- 智能硬件如何提升体验？
- 产品如何形成品牌识别？
- 产品如何从概念变成量产商品？

工业设计的核心是：

> 在用户需求、技术条件、制造工艺、商业价值和审美表达之间找到平衡。

---

### <a id="toc-6-2"></a>2. 专业定位

工业设计和产品设计关系很近，但在本科目录中定位不同：

| 专业 | 所属门类 | 偏向 |
|---|---|---|
| 工业设计 | 工学、机械类 | 产品工程、制造、结构、人机、可量产 |
| 产品设计 | 艺术学、设计学类 | 产品造型、审美、用户体验、设计表达 |

工业设计通常更强调：

- 工程基础
- 机械结构
- 材料工艺
- 人机工程
- 产品开发流程
- 制造可行性
- 工业产品系统
- 可量产设计

它不是单纯“画产品外观”，也不是机械专业的简单美术版本。

它真正做的是：

> 让产品既可用、好用、好看，又能生产、能销售、能维护。

---

### <a id="toc-6-3"></a>3. 适合人群

工业设计适合：

- 喜欢产品、工具、设备和智能硬件
- 对手绘、建模、渲染感兴趣
- 愿意理解结构、材料和工艺
- 喜欢观察用户使用行为
- 对科技产品、家电、交通工具、医疗器械感兴趣
- 愿意做模型和原型
- 能同时接受设计和工程训练
- 有较强空间想象力和审美能力

不太适合：

- 完全不喜欢工程和制造
- 只想画好看的图
- 不愿意做用户研究
- 不愿意学习建模软件
- 不愿意理解材料工艺
- 不能接受反复修改产品方案
- 不愿意做模型和测试

## <a id="toc-7"></a>七、视觉传达设计专业

### <a id="toc-7-1"></a>1. 专业特点

视觉传达设计是设计学类中最经典、应用最广的专业之一。

它研究的问题包括：

- 信息如何被视觉化？
- 品牌如何被识别？
- 海报如何吸引注意？
- 字体如何传递气质？
- 包装如何影响购买？
- 图形如何表达概念？
- 版式如何提高阅读效率？
- UI 如何让用户更容易操作？
- 广告视觉如何形成传播力？
- 新媒体视觉如何适配平台？
- 信息如何通过图表被理解？

视觉传达设计的核心是：

> 用视觉语言完成信息表达、品牌塑造和传播沟通。

---

### <a id="toc-7-2"></a>2. 专业定位

视觉传达设计不是简单“做海报”或“修图”。

它包括：

- 图形设计
- 字体设计
- 标志设计
- 品牌视觉系统
- 版式设计
- 包装设计
- 书籍设计
- 广告视觉
- 信息设计
- 导视系统
- UI 界面设计
- 动态视觉
- 新媒体视觉
- 视觉叙事

它本质上是：

> 信息、审美、传播、商业和文化之间的视觉转译。

---

### <a id="toc-7-3"></a>3. 适合人群

视觉传达设计适合：

- 喜欢平面设计、品牌、字体、插画、包装
- 对画面构图和色彩敏感
- 喜欢视觉表达和创意
- 愿意长期练习软件和版式
- 对广告、品牌、新媒体、UI 感兴趣
- 具有较强审美能力
- 能接受反复修改
- 愿意关注商业传播和用户认知

不太适合：

- 完全不喜欢视觉审美
- 不愿意学习设计软件

## <a id="toc-13"></a>十三、常见误区

### <a id="toc-13-1"></a>1. 误区一：设计专业就是画画

不准确。

画画只是表达能力的一部分。

设计更重要的是：

- 发现问题
- 分析用户
- 提出方案
- 验证方案
- 控制成本
- 组织信息
- 协调技术
- 服务商业目标
- 改善人的体验

---

### <a id="toc-13-2"></a>2. 误区二：人居设计就是室内设计

不完全正确。

人居设计可以包含室内空间，但范围更广。

它还包括：

- 社区设计
- 城市更新
- 公共空间
- 文旅空间
- 低碳人居
- 智慧空间
- 生活方式研究
- 人居系统设计

室内设计更关注具体室内空间，人居设计更强调人、空间、环境、社会和生活方式的综合关系。

---

### <a id="toc-13-3"></a>3. 误区三：工业设计就是产品外观设计

不准确。

工业设计不仅是外观，还包括：

- 用户研究
- 产品功能
- 人机工程
- 材料工艺
- 结构可行性
- 制造成本
- 交互体验
- 品牌语言
- 产品系统
- 服务流程

只会画好看的产品图，不足以成为成熟工业设计师。

---

### <a id="toc-13-4"></a>4. 误区四：视觉传达设计就是做海报

不准确。

海报只是视觉传达的一部分。

视觉传达还包括：

- 品牌系统
- 包装系统
- 字体设计
- 版式设计
- UI 界面
- 信息可视化
- 动态视觉
- 导视系统
- 广告视觉
- 新媒体视觉

---

### <a id="toc-13-5"></a>5. 误区五：AI 会让设计专业没用

不准确。

AI 会降低简单制图和初级视觉生产的门槛，但不会替代真正的设计判断。

未来更有价值的是：

- 设计策略
- 审美判断
- 用户洞察
- 品牌系统能力
- 空间体验能力
- 产品落地能力
- 跨媒介整合能力
- AI 辅助创意能力
- 与客户和团队沟通的能力

## <a id="toc-15"></a>十五、就业前景综合判断

### <a id="toc-15-1"></a>1. 人居设计就业前景

总体判断：

> 人居设计是新兴专业，适合未来人居、空间更新、文旅体验、智慧空间、社区设计和可持续设计方向。由于专业较新，就业时需要用作品集和具体项目说明自己的能力。

优势：

- 新兴交叉，发展空间大
- 适合城市更新、文旅、智慧空间和可持续方向
- 关注人的生活体验，社会价值强
- 可与建筑、景观、规划、交互、服务设计结合

挑战：

- 专业认知度仍在形成
- 需要跨学科能力
- 本科就业要靠具体作品方向
- 不能只做概念，需要落地能力

建议：

- 尽早选择空间、社区、文旅、智慧人居或可持续方向
- 多做场地调研和真实空间项目
- 强化建模、制图和方案表达
- 多关注城市更新、老龄化、低碳社区和智慧空间
- 建立完整作品集

---

### <a id="toc-15-2"></a>2. 工业设计就业前景

总体判断：

> 工业设计就业方向相对明确，适合产品、智能硬件、用户体验、交通工具、医疗产品和制造企业。它的优势是工程落地强，挑战是要求同时具备审美、建模、结构、工艺和用户研究能力。

优势：

- 产品岗位较清晰
- 可进入消费电子、家电、智能硬件、汽车、医疗器械等行业
- 作品集能直接体现能力
- 可向 UX、服务设计、产品经理方向拓展

挑战：

- 要求综合能力高
- 不能只会造型
- 初级岗位竞争较强
- 需要理解材料、工艺和制造
- 需要持续提升软件和项目能力

建议：

- 打好手绘、建模、渲染基础
- 多做产品拆解和用户研究
- 学会材料工艺和结构常识
- 做出完整产品设计项目
- 争取智能硬件、家电、消费电子、医疗产品等实习

---

### <a id="toc-15-3"></a>3. 视觉传达设计就业前景

总体判断：

> 视觉传达设计应用广、岗位多，但竞争也最激烈。初级视觉岗位容易被模板化和 AI 工具冲击，未来更需要品牌系统、UI、动态视觉、信息设计和跨媒介整合能力。

优势：

- 就业面广
- 品牌、广告、包装、UI、新媒体都有需求
- 作品转化快
- 可自由职业或工作室化发展
- 商业项目机会多

挑战：

- 学生和从业者数量多
- 初级平面岗位竞争激烈
- 低端设计容易被工具替代
- 客户修改和商业压力较大
- 需要持续提升审美和策略能力

建议：

- 不要只做单张海报，要做系统项目
- 强化品牌设计、包装设计和 UI 能力
- 学习动态视觉和信息可视化
- 建立高质量作品集
- 多做真实商业项目和实习
- 学会讲清楚设计策略，而不只是展示效果图

---

## <a id="toc-16"></a>十六、最终建议

如果你更喜欢**空间、环境、社区、城市更新、文旅体验、未来生活方式和可持续人居**，优先考虑：

> **人居设计**

如果你更喜欢**产品、智能硬件、交通工具、医疗设备、用户体验、材料工艺和产品落地**，优先考虑：

> **工业设计**

如果你更喜欢**品牌、平面、包装、字体、海报、UI、新媒体视觉和信息表达**，优先考虑：

> **视觉传达设计**

最简单的判断方式是：

- **想设计空间和生活环境**：人居设计
- **想设计产品和硬件**：工业设计
- **想设计品牌和视觉信息**：视觉传达设计
- **想做城市更新、文旅、社区**：人居设计
- **想做智能硬件、家电、汽车、医疗产品**：工业设计
- **想做品牌、包装、UI、新媒体视觉**：视觉传达设计
- **想走工程落地路线**：工业设计更合适
- **想走空间系统路线**：人居设计更合适
- **想走商业视觉路线**：视觉传达设计更合适

一句话总结：

> 人居设计偏“人如何更好地生活在环境中”，工业设计偏“产品如何更好地被使用和制造”，视觉传达设计偏“信息如何更好地被看见和传播”。三个专业都需要审美和作品集，但真正拉开差距的是调研能力、设计逻辑、软件能力、项目经验、商业理解和是否形成清晰方向。
